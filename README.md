# Silent Bell v.1.3.2

**A lightweight meditation timer with built-in log. No accounts. No tracking. No ads.**

- **Private** – Your sessions stay on your device  
- **PWA-Ready** – Install on phone or desktop  
- **Open Source** – MIT License  

---

## How to Use

1. Set the time and hit **Start**
2. After the session, details are automatically saved to your log  
3. Add a note afterward if you like, or log sessions you did without the app

**In Settings:** Adjust default duration, sound, interval bells, preparation countdown, and toggle notes on or off.

&gt; ⚠️ **iOS Important:** Do not manually turn off the screen – it will prevent the sound from ringing at the end of the session.

---

## Installation

### Option 1: Use in Browser (Quickest)

Open **[jwaseon.netlify.app](https://jwaseon.netlify.app/)** in any modern browser and start meditating immediately. No installation required.

### Option 2: Install as App (Recommended)

Silent Bell works as a Progressive Web App—install it once, use it anytime, even offline.

**iPhone / iPad (Safari):**
1. Open the site in Safari
2. Tap the **Share** button (rectangle with arrow)
3. Scroll down and tap **"Add to Home Screen"**
4. Tap **"Add"**

**Android (Chrome):**
1. Open the site in Chrome
2. Tap the menu (three dots) or look for the install prompt
3. Tap **"Install"** or **"Add to Home Screen"**

Once installed, Silent Bell launches like any native app—no browser, no address bar, just the timer.
